Go Optimizations 101

This book provides some code performance optimization tricks, tips, and suggestions, which include

  • how to consume less CPU resources.
  • how to consume less memory.
  • how to make less memory allocations.
  • how to control memory allocation places.
  • how to reduce garbage collection pressure.

The book is Go 1.20 ready now (update history).

(Please follow Go 101's official Twitter account @go100and1 to get latest Go 101 news and learn all kinds of Go knowledge.)
