현재 3권의 신간들인 Go Optimizations 101, Go Details & Tips 101 과 Go Generics 101이 출간되어 있습니다. Leanpub 서점에서 번들을 모두 구입하시는 방법이 비용 대비 효율이 가장 좋습니다.

Go에 대한 많은 정보들과 Go 101 책들의 최신 소식을 얻으시려면 Go 101 트위터 계정인 @Go100and1을 팔로잉 해주세요.

감사의 말

먼저, 모든 Go 커뮤니티에 감사드립니다. 적극적으로 응답해주는 커뮤니티가 없었다면, 이 책을 완성하지 못했을 것입니다.

Ian Lance Taylor 씨에게 특별히 감사를 전하고 싶습니다. Ian 씨는 go-nuts 그룹과 Go 프로젝트 이슈 트래커에 제가 올린 수많은 시시콜콜한 질문에 친절하게 답해주셨습니다. Ian 씨의 답변은 Go를 사용하면서 겪던 많은 혼란을 해결하는 데 도움이 되었습니다.

go-nuts 및 go-dev 그룹의 질문 글 타래에 참여해 주신 다음 분들께 감사의 인사를 전하고 싶습니다 Axel Wagner, Keith Randall, Russ Cox, Robert Griesemer, Jan Mercl, Konstantin Khomoutov, Brad Fitzpatrick, Alan Donovan, Minux Ma, Dave Cheney, Volker Dobler, Caleb Spare, Matt Harden, Roger Peppe, Michael Jones, peterGo, Pietro Gagliardi, Paul Jolly, Oleg Sidorov, and Rob 'Commander' Pike, 그 외 다른 모든 분들.

만약 제가 위에서 언급하는 것을 잊은 분이 있다면 죄송합니다. Go 커뮤니티에는 친절하고 독창성있는 고퍼들(Gophers)이 너무 많아서 몇몇 분들을 놓쳤을 수도 있습니다.

go101.org 웹사이트를 지원해주신 Bootstrap framework, jQuery library, code prettify 그리고 prism syntax highlighting JavaScript 패키지 작성자에게 감사드립니다. Go 101 전자책 제작을 맡아주신 go-epub 과 calibre 프로젝트의 작가분들께 감사드립니다.

전자책 제안에 좋은 제안을 해주신 Adam Chalkley 씨에게 감사드립니다.

Renee French씨와 Rob Pike 씨에게 특별히 감사드립니다. 이 책의 디지털 및 종이 버전의 표지에 사용된 생생한 그림은 Rob의 슬라이드 중 하나에서 복사하였습니다. Renee 씨는 그림에 사용한 상징적인 마스코트, 사랑스러운 고퍼(Gopher)의 작가입니다.

The Go 101 프로젝트는 Github 에서 호스팅됩니다. 오타, 문법 오류, 부정확한 표현, 설명 결함, 코드 버그, 끊어진 링크와 같은 모든 종류의 실수에 대한 수정 사항을 제출하여 Go 101을 개선을 돕는 것은 언제나 환영합니다.

주기적으로 Go에 대한 깊이 있는 정보를 얻고 싶다면 Go 101의 공식 트위터 계정인 @go100and1을 팔로우하거나 Go 101 슬랙 채널에j가입해주세요.

이 책의 디지털 버전은 아래와 같은 곳을 통해서 구매할 수 있습니다.

  • 아마존 킨들 스토어, (현재는 사용 불가).

  • 구글 플레이 스토어, 19.99 달러.

  • Rectangle Pushers (★★★★★), 2가지 모드와 104개 이상의 단계로 이루어진 캐주얼 퍼즐 게임
  • Let's Play With Particles, 세가지 미니 게임이 있는 캐주얼 액션 게임
